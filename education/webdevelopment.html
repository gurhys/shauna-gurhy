Naming and Naming Conventions in JavaScript

  • Give your variables meaningful names
  • Can't name variable var
  • Variable name can't start with number but can contain a number
  • No spaces in variables
  • Only able to use letters abc numbers 123 a dollar sign $ and an underscore _ in naming variables - no other symbols are valid
  • Name variables using CAMEL CASING - e.g., userScoreFinal
